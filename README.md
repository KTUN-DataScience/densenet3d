# Densenet3d
Densenet3D implementation using PyTorch

## Getting started
### Requirements
- [python 3.6 >=][pythorn] 
- CUDA [10.1][cuda10.1] or [10.2][cuda10.2]
- [pytorch 1.6.0][pytorch]
- openCV
- scipy
- Numpy

Install dependencies using requirement.txt with the following command. 
```console
pip install -r requirements.txt
```
